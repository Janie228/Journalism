Correlations Discovered Between Health Risk Factors versus Age and Income

This project is implemented to help a major metro paper analyzing the current trends that shape people's lives, as well as creating charts, graphs, and interactive elements to help readers understand the research results.

The editor wants to run a series of feature stories about the health risks facing particular demographics. This project provides the first story idea by sifting through information from the U.S. Census Bureau and the Behavioral Risk Factor Surveillance System. The current dataset is based on 2014 ACS 1-year estimates on rates of income, obesity, poverty, etc. by state.
